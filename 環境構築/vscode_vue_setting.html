VSCode + Vue 設定

環境構築

  • GitHubを設定しておく
  • VSCodeをインストール
  • ターミナル作業
    • nodejsをインストール※ CドライブのProgramFilesにインストールする
    • node --version(インストールしたnodejsのバージョンを表示)
    • npm install --global(npmをインストール)※ --globalで、どこからでもnpmを使えるようにする。書かないとinstallをしたフォルダでしか使えない。
    • npm install vue-cli --global(vueをインストール)
    • npm install nuxt --global(nuxtをインストール)

コマンド

  • npm run dev(nuxtサーバー起動)
  • npm run generate※ vueファイルをhtmlファイルに変換 & ルーティング生成
